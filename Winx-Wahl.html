4. Winx-Wahl



Einleitung:

Bei diesem Spiel kann der Nutzer herausfinden, welcher Charakter aus der Serie „Winx Club“ sein Lieblingscharakter ist.


Wie startet man den Filter ?

Der Filter bedarf keiner Startfunktion.


Welche Interaktionen gibt es?/ Wie bedient man den Filter?

Das Spiel ist folgender Maßen aufgebaut:

Es sieht aus wie ein WM-Spielplan, welcher die Achtelfinalisten, die Viertelfinalisten, die Halbfinalisten und die Finalisten anzeigt. Pro Runde ist es immer ein Zweikampf zwischen zwei Charaktere. Der Gewinner dieser Runde kommt in die nächste Runde und tritt dort gegen einen neuen Charakter an.

Durch das links und rechts Neigen des Kopfes, kann ein Charakter ausgewählt werden. Nachdem ein Charakter das Duell gewonnen hat, wird er auf der Karte um eine Position nach vorne verschoben.


Musik:

Sobald der User auf den Recording-Knopf klickt, wird das Lied „Believix“ gespielt. Ist die Aufnahme beendet, stoppt die Musik.
